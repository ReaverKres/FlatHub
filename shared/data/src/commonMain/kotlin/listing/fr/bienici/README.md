# Bien'ici (FR)

Live `BieniciListingSource` — list `realEstateAds.json` + detail `realEstateAd.json`.

Public detail URL (SPA route):

`/annonce/{location|vente}/{city-slug}/{appartement|maison}/[{n}pieces/]{id}`

Example: `https://www.bienici.com/annonce/location/paris-11e/appartement/2pieces/apimo-5517907`

Short `/annonce/location/{id}` opens the search shell — do not use.
